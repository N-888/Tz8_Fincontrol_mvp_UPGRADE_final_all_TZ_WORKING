# Tz8_Fincontrol_mvp
FinControl — аккуратный и практичный MVP-сервис для учёта личных финансов с веб-интерфейсом на Django и Telegram-ботом для быстрого ввода операций и получения сводной аналитики. Проект задуман как чистый, расширяемый и понятный MVP, который можно использовать как учебную итоговую работу, демонстрационный прототип или основу для дальнейшего развития продукта.
