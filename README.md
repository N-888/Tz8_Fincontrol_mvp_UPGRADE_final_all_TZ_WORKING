# 💼 FinControl Plus — сервис учёта личных финансов с аналитикой, экспортом отчётов и Telegram-ботом

> Современный MVP/Plus-проект для аккуратного и понятного управления личными финансами: веб-кабинет, аналитика, рекомендации, категории и подкатегории, отчёты, экспорт, Telegram-интеграция и удобный пользовательский сценарий в одном решении.

---

# ✨ О проекте

**FinControl Plus** — это практичный цифровой сервис для ведения личного финансового учёта, анализа доходов и расходов, контроля бюджета и повышения финансовой дисциплины.

Проект создан для компании **FinAware** — стартапа, который развивает удобные и доступные инструменты для повседневного управления личными финансами с упором на простоту, автоматизацию и взаимодействие через привычные каналы, включая Telegram.

## 🎯 Цель проекта

Создать удобный, наглядный и масштабируемый сервис, который помогает пользователю:

- быстро фиксировать доходы и расходы;
- видеть реальную картину финансов;
- анализировать динамику денежных потоков;
- получать советы и предупреждения;
- сохранять и экспортировать отчёты;
- использовать Telegram как быстрый канал взаимодействия с системой.

# 🧩 Что входит в проект

## Веб-кабинет
Полноценный личный кабинет пользователя с регистрацией, авторизацией, историей операций, фильтрами, аналитикой, рекомендациями, сохранёнными отчётами и настройками профиля.

## Аналитический модуль
Сводки по периодам, графики, диаграммы, баланс, выявление аномалий, предупреждения по лимитам, контроль категорий и подкатегорий.

## Telegram-бот
Интерфейс быстрого взаимодействия с системой: привязка аккаунта, команды, быстрые кнопки, сводки, добавление операций и сценарии повседневного использования.

## Экспорт отчётов
Подготовка отчётов для выгрузки и передачи в форматах **Excel / PDF**.

## Опциональные расширения
Подготовлена архитектура для подключения расширенных AI-функций и голосового ввода через Telegram как дополнительных опций, активируемых по желанию пользователя.

# 🧱 Архитектура проекта

Проект построен по модульной архитектуре:

- **Django-project** — настройки, маршруты, запуск;
- **core** — основная бизнес-логика;
- **models** — сущности данных;
- **forms** — формы ввода и настройки;
- **views** — веб-логика и пользовательские сценарии;
- **services** — аналитика, графики, Telegram-хелперы, экспорт;
- **telegram_bot** — обработчики Telegram-сценариев;
- **templates** — шаблоны интерфейса;
- **static** — стили и скрипты;
- **management commands** — запуск служебных сценариев.

### Архитектурные преимущества
- понятное разделение ответственности;
- единая база данных для сайта и Telegram-бота;
- удобное развитие и масштабирование;
- хорошая читаемость и прозрачность логики;
- удобство для демонстрации, проверки и передачи проекта.

-✅ Диаграмма распределения расходов по категориям
Реализовано как круговая диаграмма с группировкой по категориям/подкатегориям.
-✅ Фильтры: за день, неделю, месяц, год
Реализовано в дашборде и аналитическом сервисе через периодические фильтры.

-Статистика и расчёты
-✅ Подсчёт общих доходов, расходов и текущего баланса

-✅ Отчёты по категориям и периодам

▶️ Быстрый запуск проекта

1. Активировать виртуальное окружение
.\.venv\Scripts\Activate.ps1
2. Установить зависимости
pip install -r requirements.txt
3. Применить миграции
python manage.py migrate
4. Запустить сайт
python manage.py runserver

Сайт откроется по адресу:
http://127.0.0.1:8000/

5. Запустить Telegram-бота
Во втором терминале:
python manage.py runbot

⚙️ Пример .env
DJANGO_SECRET_KEY=your-secret-key-here
DJANGO_DEBUG=True
DJANGO_ALLOWED_HOSTS=127.0.0.1,localhost
DJANGO_TIME_ZONE=Europe/Moscow

TELEGRAM_BOT_TOKEN=ВАШ_ТОКЕН_ОТ_BOTFATHER
